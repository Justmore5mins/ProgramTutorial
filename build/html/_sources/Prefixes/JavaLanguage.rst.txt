要裝的東西跟Java的語法
=========================

需要裝的軟體
------------------

FRC所有需要的軟體有這些

- 寫程式的軟體: `WPILib VSCode <https://github.com/wpilibsuite/vscode-wpilib/releases/tag/v2025.3.2>`_
- 在Debug的時候看資料的軟體: `AdvantageScope <https://github.com/Mechanical-Advantage/AdvantageScope/releases/tag/v4.1.6>`_
- 比賽的時候看資料的軟體: `Elastic Dashboard <https://github.com/Gold872/elastic-dashboard/releases/tag/v2026.0.0-beta-1>`_
- 跟RoboRIO有關的東西: `FRC Game Tools <https://www.ni.com/en/support/downloads/drivers/download.frc-game-tools.html?srsltid=AfmBOop9pg2kbIhJDKAslqnmjvU7rt7d3GlyldXYAx8zQKsYjV7mmgz->`_ (他需要登入才能下載)

外部資料包(VendorDeps)的裝法：

1. 打開VSCode
2. 在側邊列找到這個長的很抽象的圖標

.. image:: https://raw.githubusercontent.com/wpilibsuite/branding/main/export/png/wpilib-icon-256.png
    :alt: WPILib icon

3. 然後找自己要的東西按install就好了(記得要連網際網路)
